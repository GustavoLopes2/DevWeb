Projetos Desenvolvidos

Projeto Space Invaders

PROJETO SPACE INVADERS, desenvolvido para a matéria de Programação Orientada a Objetos do IFPR, tem como objetivo desenvolver um jogo em Java utilizando e estudando frameworks como o Swing, JFrame e Hibernate. Com conexão com banco de dados SQL e estrutura de projeto MVC.

Portal Aluno

PROJETO PORTAL ALUNO, desenvolvido por alunos do IFPR com o objetivo de conseguir um estágio de desenvolvimento na UnitFatecie, utilizamos tecnologias de back-end e front-end para desenvolver um portal do aluno com várias funcionalidades. Utilizamos CSS, HTML, Java, Spring Boot e React.
