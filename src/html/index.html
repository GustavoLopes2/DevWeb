Sobre Mim

Olá! Meu nome é Gustavo Lopes Feitosa. Sou um desenvolvedor full-stack e atualmente trabalho na Nextage. Estou no 3º ano de Engenharia de Software no Instituto Federal do Paraná - Campus Paranavaí. Tenho experiência em Java, C#, JavaScript, React, Angular e Python.

Aqui você pode ver meus projetos, experiências e entrar em contato comigo! Sinta-se à vontade para explorar.

Projetos Desenvolvidos

Projeto Space Invaders

Criação de um jogo no estilo space invaders. Esse projeto foi realizado no IFPR com o objetivo de aprender os fundamentos da programação orientada a objetos. O resultado foi a criação de um jogo utilizando Java, Maven e banco de dados SQL.

Portal do aluno

Esse projeto foi a criação de um portal do aluno para a UniFatecie, uma competição com outras equipes valendo vaga de emprego de desenvolvedor, o projeto foi feito com React, JavaScript e CSS.

Curso Senac Java

Curso realizado pelo SENAC 2022, com introdução a Java, SQL e diagramas de classe, foi feito um projeto para a melhoria no cadastro de salas e equipamentos das salas do SENAC, para ajudar na organização de marcar hora para usar uma determinada sala.
